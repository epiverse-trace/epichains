# Log-likelihood of the length of chains with geometric offspring distribution

Log-likelihood of the length of chains with geometric offspring
distribution

## Usage

``` r
.geom_length_ll(x, prob)
```

## Arguments

- x:

  A numeric vector of chain lengths.

- prob:

  The probability of the geometric distribution with mean `1/prob`. A
  single numeric value between 0 and 1.

## Value

A numeric vector of log-likelihood values.
