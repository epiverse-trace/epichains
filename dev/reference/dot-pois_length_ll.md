# Log-likelihood of the length of chains with Poisson offspring distribution

Log-likelihood of the length of chains with Poisson offspring
distribution

## Usage

``` r
.pois_length_ll(x, lambda)
```

## Arguments

- x:

  A numeric vector of chain lengths.

- lambda:

  The rate of the Poisson distribution. A single numeric value. Must be
  positive.

## Value

A numeric vector of log-likelihood values.
