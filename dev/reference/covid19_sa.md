# COVID-19 Data Repository for South Africa

An aggregated subset of the COVID-19 Data Repository for South Africa
created, maintained and hosted by Data Science for Social Impact
research group, led by Dr. Vukosi Marivate.

## Usage

``` r
covid19_sa
```

## Format

### `covid19_sa`

A data frame with 19 rows and 2 columns:

- date:

  Date case was reported

- cases:

  Number of cases

## Source

<https://github.com/dsfsi/covid19za> Further details in
`data-raw/covid19_sa.R`.

## Details

The data is originally provided as a linelist but has been subsetted and
cleaned in `data-raw/covid19_sa.R`.
